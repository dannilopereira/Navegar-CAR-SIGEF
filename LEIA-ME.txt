Guia Visual: Hospedar e acessar Navegar-CAR-SIGEF no GitHub Pages
1️⃣ Enviar arquivos para o GitHub

Abra o Prompt de Comando na pasta:
C:\Users\edson-dannilo\Desktop\SCRIPT_NAVEGAR_CAR

Inicialize o Git (se ainda não fez):
git init

Adicione todos os arquivos:
git add .

Faça o commit:
git commit -m "Primeiro envio dos arquivos limpos"

Adicione o repositório remoto:
git remote add origin https://github.com/dannilopereira/Navegar-CAR-SIGEF.git

Configure o branch principal:
git branch -M main
Envie os arquivos para o GitHub:

git push -u origin main


2️⃣ Ativar GitHub Pages
Abra o repositório no navegador:
https://github.com/dannilopereira/Navegar-CAR-SIGEF

Clique em Configurações (ícone de engrenagem).

No menu lateral esquerdo, clique em Páginas (Pages).


Configure a fonte do site:

Filial (Branch): principal

Pasta (Folder): / (raiz)

Clique em Salvar (Save)

3️⃣ Acessar o site
URL pública gerada pelo GitHub:
https://dannilopereira.github.io/Navegar-CAR-SIGEF/

Abra no navegador → o index.html vai carregar e os JSONs serão lidos corretamente.

4️⃣ Atualizações semanais

Quando atualizar os JSONs:

Substitua os arquivos na pasta local.
No Prompt de Comando:
git add .
git commit -m "Atualização semanal"
git push
O site será atualizado automaticamente.

💡 Dica final: Sempre mantenha os JSONs na mesma pasta do index.html.

Assim, o site vai conseguir ler os dados sem erro.
